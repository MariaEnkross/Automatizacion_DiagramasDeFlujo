# ENK Generador de Unifilares

## Descripción
Esta aplicación permite a los usuarios filtrar un archivo Excel exportado desde el programa Eplan, identificando errores y uniendo conexiones coincidentes de mangueras para mejorar la claridad al generar diagramas de flujo. Finalmente, exporta los resultados en un archivo PDF. Compatible con Windows 10 y 11

## Características Principales
- Filtrado de errores en el archivo Excel de conexiones.
- Unión de conexiones de mangueras coincidentes.
- Generación de diagramas de flujo.
- Exportación de los resultados en formato PDF.

## Instalación

### Requisitos Previos
- Windows 10 o Windows 11
- Microsoft Excel
- Lector PDF

- Selección de Idioma: Elija el idioma de instalación deseado, ya sea español o inglés.
- Acuerdo de Licencia: Revise y acepte los términos del acuerdo de licencia para continuar con la instalación.
- Selección de Tareas Adicionales: Escoja si desea realizar tareas adicionales, como crear un acceso directo en el escritorio.
- Preparación para la Instalación: Verifique las opciones seleccionadas y confirme que esté listo para iniciar la instalación.
- Finalización del Proceso: Una vez completada la instalación, decida si quiere ejecutar el programa de inmediato o hacerlo más tarde.
